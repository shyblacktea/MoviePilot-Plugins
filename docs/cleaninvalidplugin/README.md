# 清理无效插件

## 核心用途

> [!NOTE]
> 用于处理 MoviePilot 已安装插件列表中残留但无法加载的记录，支持扫描、清理和重新安装。

#### **无效插件扫描**

- 扫描数据库中残留但无法加载的插件记录。
- 显示运行目录、本地插件源和异常状态。
- 帮助判断记录是应该清理，还是可以重新安装恢复。

#### **清理与重装**

- 支持批量选择无效插件。
- 支持从已安装插件列表中清理残留记录。
- 支持优先从本地插件源重新安装。
- 重新安装失败时保留原安装记录，避免误删。

#### **Vue 界面**

- 配置页与详情页使用 Vue 联邦组件重做。
- 参考路径名称格式化插件的分组卡片、紧凑表单和底部操作栏。
- 适配桌面端和手机端，手机端底部保留关闭入口。

-------

## 配置解析

### 插件处理

1. **刷新扫描**：重新读取当前 MoviePilot 插件状态，找出无法加载的记录。
2. **清理记录**：从已安装插件列表中移除选中的无效记录。
3. **重新安装**：优先从本地插件源重新安装选中的插件。
4. **关闭页面**：退出插件详情页，手机端可通过底部按钮关闭。

### 使用建议

- 清理前先确认该插件已经不再使用。
- 重新安装依赖本地插件源，找不到来源时不会强行删除原记录。
- 遇到市场缓存时，可先刷新插件市场或清理 MoviePilot 缓存。
