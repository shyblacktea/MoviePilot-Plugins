# 订阅下载增强

## 插件用途

订阅下载增强用于检测已启用电视剧订阅中，按 TMDB 播出日期计算已经超过宽限天数但仍未入库的剧集，并进一步分析未更新原因。

诊断结果分为：

- 暂无 PT 资源
- 疑似识别问题
- 订阅规则拦截
- 可下载
- 搜索失败

## 配置说明

- 启用：开启后按 Cron 定时扫描。
- 宽限天数：按 `TMDB 单集播出日期 + N 天 <= 当前日期` 判断是否触发。
- 二级分类：读取 MoviePilot 二级分类策略中的电视剧分类，默认全选；订阅自身无分类时会按 TMDB 信息识别分类。
- 分类站点范围：每个二级分类可指定搜索站点；未指定时使用 MoviePilot “设置-搜索&下载”里的搜索站点，站点变更后刷新配置页或下次扫描会同步。
- Telegram 独立通知：每部剧单独发送一条通知，菜单状态互不影响。
- 允许 TG 修改订阅规则：开启后 Telegram 按钮可生成规则预览并确认写入。

## Telegram 操作

每部剧的通知包含剧名、季号、缺失集、播出日期、诊断原因、候选资源数量和搜索站点范围。

按钮包括：

- 下载：展示候选资源并提交下载任务。
- 调整规则：按候选资源生成平台关键词或站点建议，通过 Telegram 菜单选择后生成 include 预览。
- 返回：回到上一级菜单。
- 忽略本次：跳过当前剧当前缺失集组合的重复提醒。
- 关闭：结束本次交互。

## 安全与回退

- 规则修改必须先生成预览。
- 确认后才写入订阅规则。
- 每次写入都会记录旧值、新值、来源和时间。
- 插件不自动删除订阅，不自动暂停订阅。

## 更新日志

### v0.1

- 初版支持 TMDB 播出日期停更检测。
- 支持读取 MoviePilot 二级分类策略，订阅自身无分类时按 TMDB 信息识别分类。
- 支持按二级分类配置 PT 搜索站点范围。
- 分类站点范围同步 MoviePilot “设置-搜索&下载”里的搜索站点。
- 支持 PT 搜索诊断、Telegram 独立通知和 Vue 配置详情页。
- Telegram 规则调整改为按候选资源动态选择平台关键词或站点建议。
