# 订阅下载增强

`SubscribePlus` 用于检测电视剧订阅：当 TMDB 单集播出时间超过设定天数后仍未下载入库时，插件会优先复用 MoviePilot 原生订阅搜索结果，并分析原因是暂无资源、识别问题，还是订阅规则没有命中。

## 主要功能

- 按 `TMDB 单集播出日期 + N 天 <= 当前日期` 判断是否触发检测。
- 支持勾选二级分类，默认读取 MoviePilot 分类策略并全选电视剧二级分类。
- 优先触发 MoviePilot 原生订阅搜索，MP 可直接匹配下载时不重复通知。
- 插件 PT 搜索范围可在配置页独立选择，并通过 Telegram 通知里的 `PT范围搜索` 手动触发。
- 诊断候选只保留最近缺失集数，例如已入库到 E07、TMDB 已播到 E09，则只分析 E08/E09。
- 每部剧独立发送 Telegram 通知，互不影响。
- Telegram 内可查看候选资源、触发 MP 订阅搜索或 PT 范围搜索、选择候选下载、返回上级菜单或结束交互，结束后会尝试删除当前通知消息。
- 支持根据候选资源添加官组或平台关键词到订阅包含规则。
- 官组来源会尝试读取 MoviePilot 内置/自定义识别词，以及订阅自身的自定义词。
- 支持自定义识别词工具，可在页面或 Telegram `/ci` 命令中自动/手动写入单条识别词。
- 支持最终集整季包清理：当最终集来自整季包时，可清理旧拆包转移记录，或同时删除旧拆包源文件。
- 支持最终集命中整季包时，将 qBittorrent 当前种子的全部文件自动设为下载，便于补齐 E01-E13 这类全包。

## 诊断结果

插件会尽量把未更新原因归类为：

- 暂无 PT 资源
- 疑似识别问题
- 订阅规则拦截
- 可下载
- 搜索失败

## Telegram 交互

通知会包含剧名、季号、缺失集、播出日期、诊断原因、候选资源数量和搜索站点范围。

常用按钮：

- `下载`：普通通知会触发 MP 原生订阅搜索；PT 范围搜索结果会展示候选资源并提交下载任务。
- `PT范围搜索`：按插件配置的 PT 站点范围手动补充搜索当前缺失集。
- `调整规则`：基于候选资源生成「添加官组」或「添加平台」建议。
- `返回`：回到上一级菜单。
- `忽略本次`：跳过当前剧当前缺失集组合的重复提醒。
- `结束`：结束本次交互，并尝试删除当前 Telegram 通知消息。

规则修改会先生成预览，确认后才写入订阅规则。插件不会自动删除订阅，也不会自动暂停订阅。

## 自定义识别词

数据页提供自定义识别词工具：

- 自动处理：填写媒体文件名后，插件调用 AI 识别目标并写入一条强制 TMDB 识别词。
- 手动处理：填写媒体文件名、`movie/tv` 和 TMDB ID 后，插件写入一条强制 TMDB 识别词。
- 写入时只把新增识别词插入到 MoviePilot 自定义识别词最顶行，不会改动旧规则。
- 写入后会刷新解析缓存并再次识别验证，结果记录会展示成功或失败原因。

Telegram 可发送 `/ci 媒体文件名` 打开自动/手动处理菜单。

## 待处理诊断（/sp）

`/sp` 用于在 Telegram 里查看当前待处理的诊断并进入单剧详情，无需等待定时扫描通知。

- 直接发送 `/sp`：列出当前所有待处理诊断的剧集，可逐个进入。
- 进入单剧详情后，可查看缺失集、诊断原因、候选资源，并触发 `PT范围搜索`、选择候选下载、调整规则等操作。
- 候选资源按钮会显示 `站点｜官组｜平台｜清晰度｜做种数`，例如 `猫站｜HHWeb｜Baha｜1080p｜做种34`，便于同一站点多条资源时快速区分，不必再回看通知详情。

## 候选下载缓存

候选资源的下载信息会缓存到本地，避免插件重载或 MoviePilot 重启后，`/sp` 里选择候选下载时提示「候选下载上下文已失效」。

- 每条候选会持久化种子直链等下载所需的最小字段，并在到期后自动清理。
- 缓存有效期由 `候选缓存天数` 控制，默认 3 天；设为 0 表示关闭缓存（回到仅内存、重启即失效的旧行为）。
- 内存上下文丢失时，插件会先用本地缓存重建下载；缓存也不存在时才回退到 MP 原生订阅搜索。

## 配置说明

- `启用`：开启后按 Cron 定时扫描。
- `宽限天数`：例如 TMDB 播出日期为 7 月 3 日，宽限天数填 1，则 7 月 4 日仍未下载会触发。
- `订阅部数通知上限`：单次扫描最多诊断并通知多少部订阅，避免一次推送过多。
- `二级分类`：可多选，例如日番、日韩剧；默认全选可识别的电视剧二级分类。
- `PT搜索范围`：插件自己的 PT 站点范围，用于 Telegram 二段 `PT范围搜索`。
- `最终集整季包清理`：默认关闭；可选择仅删除旧拆包转移记录，或删除旧拆包转移记录和源文件。该功能不会删除媒体库目标文件。
- `qB 整季包全选下载`：默认关闭；开启后，当最终集来自整季包时，会把该 qBittorrent 种子的所有文件优先级设为下载，可与旧记录清理分开使用。
- `允许 TG 修改订阅规则`：开启后 Telegram 按钮可生成并确认写入包含规则。
- `候选缓存天数`：候选下载信息的本地缓存有效期，默认 3 天；设为 0 关闭缓存。
- `通知方式`：每部剧单独发送 Telegram 通知。
